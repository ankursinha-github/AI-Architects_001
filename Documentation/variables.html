Custom Variable Declarations: blo, glo, and fix

In our new programming language, we introduce custom keywords to declare variables with specific scopes and mutability. These keywords are designed to simplify the declaration process while making the code more readable and intuitive.

VisionScript Variables can be declared in 4 ways:

  • Automatically
  • Using blo
  • Using glo
  • Using fix

1. blo (Block-scoped, Mutable)

Equivalent to:

let

Description: Use

blo

to declare variables that are block-scoped and mutable. These variables are confined to the block in which they are defined and can be reassigned within that block.

Pyonic

                    
                        blo count = 10;
                        if (count > 5) {
                            blo message = "Count is greater than 5";
                            console.log(message);  // Output: Count is greater than 5
                        }
                        console.log(message);  // Error: message is not defined (because it's block-scoped)
