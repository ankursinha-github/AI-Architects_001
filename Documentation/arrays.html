Arrays

In VisionScript Language, arrays are a fundamental data structure that allows you to store a collection of elements in a single variable. Arrays can hold elements of any type, including numbers, strings, objects, and even other arrays. They are dynamic, meaning you can add, remove, or modify elements at any time.

Array Methods

  1. array.mode() - finds the most frequent element in an array
  2. array.median() - calculates the median of an array
  3. array.quantile() - calculates a quantile of an array
  4. array.correlation() - calculates the correlation between two arrays
  5. array.covariance() - calculates the covariance between two arrays
